## 📌 Sobre o Projeto

Este é o meu projeto de portfólio pessoal, desenvolvido como trabalho acadêmico. O objetivo principal é consolidar meus conhecimentos em desenvolvimento web front-end e criar um espaço dinâmico para apresentar minha trajetória, projetos, habilidades e formas de contato.

O design foi pensado para ser **moderno, limpo e totalmente responsivo**, adaptando-se perfeitamente a diferentes tamanhos de tela (computadores, tablets e smartphones).

---

## 🚀 Funcionalidades e Diferenciais

* **Design Responsivo:** Layout fluido que se adapta a qualquer dispositivo.
* **Menu de Navegação Dinâmico:** Navegação intuitiva estruturada lado a lado.
* **Barra de Redes Sociais Fixa:** Links importantes posicionados estrategicamente no canto inferior esquerdo para fácil acesso, mantendo a identidade visual original.
* **Seção de Projetos:** Galeria para expor meus principais trabalhos e códigos.
* **Formulário de Contato:** Área dedicada para mensagens e conexões profissionais.

---

## 🛠️ Tecnologias Utilizadas

O projeto foi construído utilizando tecnologias web nativas, sem a necessidade de frameworks externos, focando na semântica e na performance:

* **HTML5:** Estruturação semântica de todo o conteúdo.
* **CSS3:** Estilização, layout (Flexbox/Grid), responsividade e efeitos visuais.
* **JavaScript (ES6):** Manipulação do DOM, comportamento do menu e interatividade.
